1. **Introduction Section**:

   - Explain what tools and evidence are available
   - Provide the first investigative direction

2. **Investigation Steps** (for each step):

   - Command or technique suggestion
   - Expected finding description

4. **Conclusion**:
   - Summary of findings
   - Connection to broader cybersecurity concepts
   - Assessment questions

- ids_investigation: Analysing IDS/IPS alerts
